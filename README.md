# Bennett Court Booking System

A web-based booking system for university sports courts.

This project allows students to book sports courts while enforcing booking rules such as operating hours and time-slot restrictions.

## Features

- Bennett University themed UI
- Login system restricted to Bennett email domain
- Multiple sports supported
- Court booking with time validation
- Booking limited to operating hours (7 AM – 11 PM)
- Hourly booking slots only
- Court availability display

## Sports Available

- Basketball
- Badminton
- Volleyball
- Table Tennis
- Lawn Tennis
- Pickleball

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

Database
- MongoDB

## How to Run

1. Clone the repository

```
git clone https://github.com/chica07/bennett-court-booking-system.git
```

2. Install dependencies

```
npm install
```

3. Run the server

```
node backend/server.js
```

4. Open in browser

```
frontend/index.html
```
